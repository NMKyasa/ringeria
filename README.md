# 💍 Ringeria E-Commerce Platform

Ringeria is a modern e-commerce platform designed for buying and selling premium rings, including wedding rings, engagement rings, and fashion rings in Kampala, Uganda.

The platform showcases a responsive design, product categories, a functional shopping cart, and a simple checkout system integrated with Mobile Money.

- 📍 Location: Pioneer Mall, Shop PD100, Kampala, Uganda

---

## 🚀 Features

- 🛍 Product categories (Wedding Rings, Engagement Rings, Gold, Silver, Tungsten)
- 🛒 Shopping cart system (Add & Remove items)
- 💳 Checkout system with Mobile Money payment option
- 📱 Fully responsive (mobile-friendly design)
- 📑 FAQ and Contact sections
- 🗺 Sitemap page for navigation structure
- 🎨 Modern UI using Bootstrap template

---

## 🛠 Technologies Used

- HTML5  
- CSS3 (Custom styling + Bootstrap)  
- JavaScript  
- PHP (Core PHP – no framework)  
- Bootstrap 5  
- Swiper.js (for sliders)

---

## ⚙️ How to Run the Project

1. Install XAMPP (or any local server)
2. Place the project folder inside:

'htdocs/'

3. Start Apache from XAMPP
4. Open your browser and go to: http://localhost/ringeria


---

## 💰 Payment Method

Payments are handled via **Mobile Money**:

- Users are prompted to send payment to the provided number  
- Confirmation is done via WhatsApp or phone call
